# FloatNote Low-Fi Screens

## 목적

이 폴더는 FloatNote의 `초기 시안`을 담는다.

- 현재 기준의 최신 방향은 루트 문서인 `DESIGN_PRINCIPLES.md`, `PRODUCT_PLAN.md`, `UX_FLOW.md`, `WIREFRAMES.md`가 소스 오브 트루스다.
- 이 폴더의 자산은 `content-first 리디자인 이전` 탐색 결과로 보관한다.

## 파일

- `floatnote-lofi.html`: 브라우저에서 열 수 있는 저해상도 화면 시안
- `floatnote-lofi.png`: 상단 기준 빠른 미리보기 캡처
- `floatnote-lofi-full.png`: 전체 화면 캡처

## 포함된 화면

1. 메인 플로팅 창: 기존 노트
2. 메인 플로팅 창: 새 노트
3. 메인 플로팅 창: 첫 노트 왼쪽 경계
4. 첫 실행 안내 오버레이
5. 설정 시트

## 현재 해석

- 당시 시안은 구조 검증용으로는 의미가 있었다.
- 하지만 현재 방향인 `조용한 문서`, `얇은 크롬`, `설명 라벨 제거` 기준에서는 더 이상 최신안이 아니다.
- 따라서 구현과 새 시안 작업은 루트 문서를 기준으로 다시 진행한다.
