# FloatNote Low-Fi Screens

## 목적

이 폴더는 FloatNote의 MVP 화면을 저해상도 수준으로 검증하기 위한 시안을 담는다.

- 핵심 검증 대상은 `본문 집중도`, `상단/하단 바 밀도`, `새 노트 전환`, `첫 실행 안내`, `설정 진입`이다.
- 색, 브랜딩, 마이크로 인터랙션은 의도적으로 후순위로 둔다.
- 인라인 마크다운 렌더링은 2단계 목표이므로 이 시안에서는 원문 입력 상태를 우선 보여준다.

## 파일

- `floatnote-lofi.html`: 브라우저에서 열 수 있는 저해상도 화면 시안
- `floatnote-lofi.png`: 상단 기준 빠른 미리보기 캡처
- `floatnote-lofi-full.png`: 전체 화면 캡처

## 포함된 화면

1. 메인 플로팅 창: 기존 노트
2. 메인 플로팅 창: 새 노트
3. 메인 플로팅 창: 첫 노트 왼쪽 경계
4. 첫 실행 안내 오버레이
5. 설정 시트

## 확인 포인트

- 창이 떴을 때 가장 먼저 본문이 보이는가
- 현재 노트 위치가 과하지 않게 보이는가
- 하단 `Created`, `Updated`가 조용하지만 충분히 읽히는가
- 새 노트 생성이 별도 모달 없이 이해되는가
- 설정 화면이 메인 흐름을 방해하지 않을 정도로 가벼운가
